🎯 Overview

UOS TV is a WebOS-style TV operating system built on Armbian (Debian-based Linux for ARM64). The entire system stack — from IPC bus to OTA updater — is written in Rust, with a HTML/CSS/JS shell (Luna UI) rendered via WPE WebKit compositor (Cog).

Philosophy: Minimal, fast, statically-linked binaries. No systemd, no dbus, no Node.js runtime. All IPC through stardust — a custom CBOR-over-Unix-socket message broker.

🚀 QEMU Boot (Working!)

VERIFIED 2026-06-03 All 11 UOS services start successfully inside QEMU aarch64 VM.

Boot Flow

1. Alpine kernel (vmlinuz-virt 6.12.1) boots on QEMU virt machine
2. Alpine initramfs loads virtio-blk, mounts root=/dev/vda (ext2)
3. switch_root → /sbin/init (custom UOS init script)
4. Init mounts proc/sysfs/devtmpfs/tmpfs, configures network
5. stardustd starts → IPC broker on /run/uos/bus.sock + WS on :9090
6. logd starts → /run/uos/log.sock
7. All 9 remaining services start in parallel
8. "UOS TV Ready" → drops to shell

Boot Command

qemu-system-aarch64 -machine virt,highmem=off -cpu cortex-a57 -smp 2 -m 512 \
  -kernel build/kernel/alpine-vmlinuz \
  -initrd build/kernel/alpine-initramfs \
  -append "console=ttyAMA0,115200 root=/dev/vda rw rootfstype=ext2 init=/sbin/init" \
  -drive file=build/uos-boot-final.img,format=raw,if=none,id=drive0 \
  -device virtio-blk-device,drive=drive0 \
  -netdev user,id=net0,hostfwd=tcp::2222-:22,hostfwd=tcp::8080-:9090 \
  -device virtio-net-device,netdev=net0 \
  -nographic

⚡ Developer Quickstart

Prerequisites

brew install qemu genext2fs     # macOS
docker run --privileged ...     # Docker for cross-compilation
rustup target add aarch64-unknown-linux-musl

Build & Boot (from scratch)

# 1. Cross-compile semua service
make docker-build-all

# 2. Download Alpine kernel + rootfs
make fetch-kernel

# 3. Overlay UOS binaries ke Alpine rootfs
./scripts/overlay-alpine.sh build/alpine-rootfs-extracted

# 4. Buat ext2 disk image
genext2fs -d build/alpine-rootfs-extracted -b $((256*1024)) -N 8192 build/uos.img

# 5. Boot!
qemu-system-aarch64 \
  -machine virt -cpu cortex-a57 -smp 2 -m 512 \
  -kernel build/kernel/alpine-vmlinuz \
  -initrd build/kernel/alpine-initramfs \
  -append "console=ttyAMA0,115200 root=/dev/vda rw rootfstype=ext2 init=/sbin/init" \
  -drive file=build/uos.img,format=raw,if=none,id=drive0 \
  -device virtio-blk-device,drive=drive0 \
  -nographic

Dev Mode (macOS native)

# Run full UOS stack locally (no cross-compile, no QEMU)
./scripts/dev-run.sh

# This starts:
#   stardustd  →  Unix socket + WebSocket bridge ws://127.0.0.1:9090
#   logd       →  /tmp/uos-log.sock
#   9 services →  netmd, devmand, dispald, inputd, audiod, powermand, notifd, pkgd, otad
#   Luna UI    →  http://localhost:8080
